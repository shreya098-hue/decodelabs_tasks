# AI Learning Platform 🚀

A modern and responsive AI Learning Platform built using HTML, CSS, and JavaScript. This project provides students with a structured roadmap to learn Artificial Intelligence, Machine Learning, Deep Learning, and Generative AI.

## 📌 Features

* Responsive Navigation Bar
* Hero Section with Call-to-Action
* AI Course Listings
* AI Learning Roadmap
* Feature Highlights
* Student Testimonials
* Pricing Plans
* Contact Form
* Smooth Scrolling Navigation
* Interactive JavaScript Functionality
* Fully Responsive Design

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript

## 📚 Sections Included

* Home
* Courses
* Roadmap
* Features
* Testimonials
* Pricing
* Contact

## 🎯 Learning Outcomes

Through this project, I practiced:

* Frontend Web Development
* Responsive Web Design
* CSS Flexbox & Grid
* DOM Manipulation
* Event Handling in JavaScript
* UI/UX Design Principles

## 🚀 How to Run

1. Clone the repository
2. Open the project folder
3. Run `index.html` in your browser
